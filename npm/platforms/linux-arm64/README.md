# @codex-mcp-rs/linux-arm64

Platform-specific binary package for [codex-mcp-rs](https://github.com/missdeer/codex-mcp-rs) (Linux arm64).

## Platform

- **OS**: Linux
- **Architecture**: arm64 (aarch64)

## Usage

This package is automatically installed as an optional dependency when you use the main package:

```bash
npx @missdeer/codex-mcp-rs
```

Or install globally:

```bash
npm install -g @missdeer/codex-mcp-rs
```

**Do not install this package directly.** Use the main `@missdeer/codex-mcp-rs` package instead, which will automatically select the correct platform binary.
